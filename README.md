## ❓ FAQ

**How long does deployment take?**
A few minutes — server boot, DNS, Let's Encrypt cert, and agent install. Every step is streamed live.

**How does the refund work?**
Delete a server → Hermesita computes unused hours → Paddle issues the adjustment the same day.

**Where are servers hosted?**
Helsinki, Finland — renewable-energy EU region. More regions coming.

**Is my data isolated?**
Yes. Every server is a dedicated VM. Nothing shared at OS or process level.

**Can I use my own domain?**
Not yet — `*.hermesita.com` subdomains only. Custom domains are on the roadmap.
