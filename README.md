## 🤖 Engines

### Hermes Agent
An open-source AI agent runtime for building autonomous workflows, multi-step task chains, and tool-augmented assistants on your own infrastructure. Hermesita installs Hermes on a dedicated AMD server, wires up a `name.hermesita.com` subdomain with Let's Encrypt HTTPS, and provides you the dashboard URL and encrypted login credentials — no SSH required.

**Highlights:**
- One-click deployment (server, DNS, TLS, runtime in a single live flow)

## ❓ FAQ

**How long does deployment take?**
Usually a few minutes end-to-end — server boot, cloud-init, DNS propagation, Let's Encrypt cert issuance, and agent software install. Each step is streamed live on the dashboard.

**How does the refund work?**
When you delete a server, Hermesita computes the unused portion of the month down to the hour and issues a Paddle adjustment. It appears in your invoices the same day.

**Where are servers hosted?**
Helsinki, Finland — a low-latency EU region powered by renewable energy. More regions are on the roadmap.

**Is my data isolated?**
Yes. Every server is a dedicated VM running only your agent instance. Nothing is shared at the OS or process level.

**Can I use my own domain?**
Not yet — all servers get a `*.hermesita.com` subdomain. Custom domains are on the roadmap.

**What happens to my data when I delete a server?**
The underlying machine and disk are destroyed. Export anything you need **before** deletion. Hermesita does not snapshot or back up server contents by design.
